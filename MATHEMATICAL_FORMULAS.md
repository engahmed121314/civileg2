# 📐 توثيق المعادلات الرياضية والحسابات الهندسية

## 🎯 نظرة عامة

هذا الملف يوثق جميع المعادلات الرياضية المستخدمة في التطبيق مع شرح مفصل ومراجع هندسية.

---

## 🔢 معادلات تصميم الكمرات (Beams)

### 1. حساب العزم (Bending Moment)

#### المصادر:
```
ECP 203 (Egyptian Code)
ACI 318 (American Code)
SBC (Saudi Building Code)
```

#### المعادلات حسب الكود:

**أ. الكود المصري (ECP 203):**
```
Mu = (1.4 * DL + 1.6 * LL) * L² / 8
حيث:
  Mu = العزم المصمم (kN·m)
  DL = الحمل الميت (kN/m)
  LL = الحمل الحي (kN/m)
  L = الطول (m)
  المعاملات: 1.4 للحمل الميت، 1.6 للحمل الحي
```

**ب. الكود الأمريكي (ACI 318):**
```
Mu = (1.2 * DL + 1.6 * LL) * L² / 8
معاملات مختلفة: 1.2 و 1.6
```

**ج. الكود السعودي (SBC):**
```
Mu = (1.2 * DL + 1.6 * LL) * L² / 8
نفس الكود الأمريكي
```

**د. القيمة المسموحة:**
```
المعادلة المستخدمة في الكود:
Mu_allowed = φ * Mn
حيث:
  φ = معامل التخفيض:
      - للكود المصري: 0.80
      - للكود الأمريكي: 0.90
  Mn = القدرة الاسمية (Nominal Capacity)
```

### 2. حساب مساحة الحديد المطلوبة

#### المعادلة العامة:
```
ρ = (0.85 * fc * ω) / fy

حيث:
  ρ = نسبة التسليح
  ω = معامل المقاومة (Resistance Factor)
  fc = مقاومة الخرسانة (MPa)
  fy = مقاومة الحديد (MPa)
  
ω = 1 - √(max(0.0, 1 - (2 * Ru) / (0.85 * fc)))

حيث:
  Ru = Mu * 10^6 / (b * d²)
  b = عرض الكمرة (mm)
  d = العمق الفعال (mm) = h - 50 (تقريباً)
```

#### الحد الأدنى للتسليح:
```
As_min = max(ρ * b * d, 0.225 * √fc / fy * b * d)
```

### 3. حساب قدرة المقطع (Capacity)

#### معادلة القدرة:
```
Mn = φ * As * fy * (d - a/2)

حيث:
  Mn = قدرة المقطع (kN·m)
  φ = معامل التخفيض
  As = مساحة الحديد المدرفلة (mm²)
  fy = مقاومة الحديد (MPa)
  a = عمق منطقة الضغط = (As * fy) / (0.85 * fc * b)
  d = العمق الفعال (mm)
```

#### التحقق من الأمان:
```
✓ آمن إذا: Mn ≥ Mu
✗ غير آمن إذا: Mn < Mu
```

### 4. حساب تشقق الخرسانة (Cracking)

#### رابطة المسافة للتشقق:
```
حد أقصى للتشقق (mm):
  - عادي: 0.3 mm
  - بيئة عدوانية: 0.2 mm
  - تجميع مياه: 0.1 mm
```

## 🏗️ معادلات تصميم الأعمدة (Columns)

### 1. حساب القدرة الاسمية

#### المعادلة الأساسية:
```
Pn = 0.85 * fc * (Ag - As) + fy * As

حيث:
  Pn = القدرة الاسمية (kN)
  fc = مقاومة الخرسانة (MPa)
  Ag = المساحة الكلية للمقطع (mm²)
  As = مساحة الحديد (mm²)
  fy = مقاومة الحديد (MPa)
  
القدرة المصممة:
  φ * Pn = معامل التخفيض * القدرة الاسمية
  
معاملات التخفيض:
  - عامود عادي (Tied): φ = 0.65
  - عامود حلزوني (Spiral): φ = 0.75
```

### 2. نسبة التسليح للعامود

#### الحدود المسموحة:
```
الحد الأدنى: ρ_min = 1% = 0.01
الحد الأقصى: ρ_max = 8% = 0.08 (مع الاعتبارات الإنشائية)

نسبة التسليح = As / Ag
```

### 3. حساب الحديد الملف (Ties/Hoops)

#### الحد الأدنى للقطر:
```
قطر الحديد الملف (Tie Diameter):
  - الحد الأدنى = 10 mm
  - يجب ألا يقل عن 1/4 من قطر الحديد الطولي
```

#### المسافة بين الأحاط:
```
المسافة القصوى = min(
  - 48 * قطر الحديد الملف
  - 16 * قطر الحديد الطولي
  - 300 mm
)
```

## 🔲 معادلات تصميم البلاطات (Slabs)

### 1. حساب العزم في البلاطات أحادية الاتجاه

#### المعادلة:
```
Mu = (1.4 * DL + 1.6 * LL) * L² / 8

حيث:
  L = الامتداد الأقصر (أحادي الاتجاه)
  أو L = الامتداد في كل اتجاه (ثنائي الاتجاه)
```

### 2. سمك البلاطة الأدنى

#### من الكود المصري:
```
h_min = L / 20  (للبلاطات الحرة)
h_min = L / 30  (للبلاطات المستمرة)

حيث:
  h_min = السمك الأدنى المسموح به
  L = الامتداد
```

### 3. نسبة التسليح

#### الحدود المسموحة:
```
ρ_min = 0.0012  (1.2 للألف)
ρ_max = 0.02    (2%)

الحد الأدنى = ρ_min * b * h
حيث b = 1000 mm (لكل متر عرض)
```

## 🏠 معادلات تصميم القواعد (Footings)

### 1. حساب الضغط على التربة

#### المعادلة:
```
q = P / A ± M / I

حيث:
  q = الضغط على التربة (kPa)
  P = القوة المحورية (kN)
  A = مساحة القاعدة (m²)
  M = العزم (kN·m)
  I = عزم القصور الذاتي
```

### 2. حساب سمك القاعدة

#### من شرط القص (Shear):
```
السمك = √(Pu / (φ * √fc * b))

حيث:
  Pu = الحمل المصمم (kN)
  φ = معامل التخفيض = 0.75
  fc = مقاومة الخرسانة (MPa)
  b = عرض القاعدة (mm)
```

### 3. التحقق من الثبات

#### معامل الأمان من الانزلاق:
```
FSslide = (P * tan(φ) + c * A) / H

حيث:
  P = الحمل الرأسي
  φ = زاوية الاحتكاك
  c = تماسك التربة
  A = مساحة التلامس
  H = الحمل الأفقي
  
FSslide ≥ 1.5 (الحد الأدنى المسموح)
```

## 🪵 معادلات تصميم السلالم (Stairs)

### 1. الشروط الهندسية

#### الشروط الأساسية:
```
2 * R + T = 600-650 mm

حيث:
  R = القائمة (Riser) = 150-180 mm
  T = النافذة (Tread) = 250-300 mm
  
شروط:
  R_min = 150 mm
  R_max = 200 mm
  T_min = 250 mm
  T_max = 350 mm
```

### 2. حساب العزم في السلالم

#### النموذج:
```
السلم يعتبر جسر مائل:
  L_eff = L_horizontal / cos(θ)
  
θ = arctan(R / T)

ثم يتم حساب العزم كالجسور العادية
```

## 💧 معادلات تصميم خزانات المياه (Water Tanks)

### 1. الحمل الهيدروستاتيكي

#### الضغط:
```
P = γ * h

حيث:
  P = الضغط (kPa)
  γ = وزن الماء = 10 kN/m³
  h = العمق (m)
```

### 2. القوة على الجدار

#### حساب القوة:
```
F = ∫₀ʰ P * dA = (1/2) * γ * h * L * h

حيث:
  L = طول الجدار
  h = ارتفاع الماء
  
F = (1/2) * 10 * h² * L
```

### 3. موقع القوة المحصلة

#### المسافة من السطح:
```
ȳ = h / 3

أي أن القوة تؤثر على ارتفاع h/3 من سطح الماء
```

## 🏔️ معادلات تصميم حوائط السند (Retaining Walls)

### 1. الضغط الجانبي للتربة

#### معامل الضغط النشط:
```
Ka = tan²(45° - φ/2)

حيث:
  φ = زاوية الاحتكاك الداخلي للتربة
  
مثال: إذا φ = 30°
  Ka = tan²(45° - 15°) = tan²(30°) = 1/3 ≈ 0.33
```

#### الضغط على الجدار:
```
σ_a = Ka * γ * h

حيث:
  σ_a = الضغط النشط (kPa)
  γ = وزن التربة (kN/m³)
  h = الارتفاع (m)
```

### 2. القوة المحصلة على الجدار

#### حساب القوة الكلية:
```
Pa = (1/2) * Ka * γ * h²

حيث:
  Pa = القوة النشطة (kN/m)
  
نقطة التطبيق: h/3 من القاع
```

### 3. التحقق من الثبات

#### معامل الأمان من الانزلاق:
```
FSslide = (W * tan(φ_base) + c_base * A) / Pa

حيث:
  W = وزن الجدار والتربة فوقه
  φ_base = زاوية الاحتكاك بين الجدار والتربة
  c_base = التماسك
  A = المساحة
  Pa = الضغط النشط
  
FSslide ≥ 1.5 (الحد الأدنى)
```

#### معامل الأمان من الانقلاب:
```
FSoverturning = M_resisting / M_overturning

حيث:
  M_resisting = العزم المقاوم (وزن الجدار)
  M_overturning = العزم المقلب (ضغط التربة)
  
FSoverturning ≥ 1.5 (الحد الأدنى)
```

## ✅ التحقق من النتائج

### معايير الأمان:

```
للكود المصري (ECP 203):
✓ معامل الأمان من الانزلاق: ≥ 1.5
✓ معامل الأمان من الانقلاب: ≥ 1.5
✓ الضغط على التربة: ≤ قدرة التحمل

للكود الأمريكي (ACI 318):
✓ معامل الأمان: ≥ 1.4
✓ معامل الأمان (مراجعة): ≥ 1.5

للكود السعودي (SBC):
✓ معامل الأمان: ≥ 1.35
✓ معامل الأمان (مراجعة): ≥ 1.5
```

### تحذيرات:

```
⚠️ تحذير إذا:
  - Capacity < Required Load (غير آمن)
  - Deflection > Limit (انحراف زائد)
  - Cracking > Limit (تشقق زائد)
  - Shear Stress > Max (قص زائد)
  - Settlement > Limit (هبوط زائد)
```
