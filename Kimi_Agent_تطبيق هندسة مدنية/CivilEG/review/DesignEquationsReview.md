# مراجعة المعادلات الحسابية والتصميمية - تطبيق Civil EG

## 1. معادلات الكمرة (Beam Design)

### 1.1 حساب العمق الفعال (Effective Depth)
```kotlin
// d = h - cover - stirrup_dia - bar_radius
d = h - 40 - 10 - (main_bar_dia / 2)
```
- **الكود المصري (ECP 203)**: الغطاء الخرساني 25-50 مم حسب البيئة
- **ACI 318**: الغطاء الخرساني 40 مم للأعمدة والكمرات

### 1.2 حساب معامل الربط (Ru)
```kotlin
Ru = Mu * 10^6 / (b * d^2)  // N/mm²
```
- **الوحدات**: Mu (kN.m) → N.mm, b,d (mm)
- **حدود Ru**: 
  - ECP: Ru_max = 0.67 * fcu * q_max * (1 - 0.5*q_max)
  - ACI: Ru_max = 0.85 * fc' * β1 * (εu/(εu+εy)) * (1 - 0.5*β1*(εu/(εu+εy)))

### 1.3 حساب نسبة التسليح (ρ)
**الكود المصري (ECP 203-2018):**
```kotlin
q = 1 - sqrt(1 - (2 * Ru) / (0.67 * fcu))
ρ = (0.67 * fcu * q) / fy
```

**ACI 318-19:**
```kotlin
ω = 1 - sqrt(1 - (2 * Ru) / (0.85 * fc'))
ρ = (0.85 * fc' * ω) / fy
```

**الكود السعودي (SBC 304):**
- يستخدم نفس معادلات ACI مع معاملات تعديل بسيطة

### 1.4 مساحة الحديد المطلوبة
```kotlin
As_req = ρ * b * d  // mm²
```

### 1.5 التحقق من نسبة التسليح
```kotlin
// الحد الأدنى (ECP)
ρ_min = max(0.225*sqrt(fcu)/fy, 1.3/fy)

// الحد الأقصى (ECP)
ρ_max = 0.4 * (fcu/fy) * (0.003/(0.003+0.004))

// ACI
ρ_min = max(0.25*sqrt(fc')/fy, 1.4/fy)
ρ_max = 0.75 * ρ_balanced
```

### 1.6 حساب قدرة تحمل العزم (Moment Capacity)
```kotlin
a = (As * fy) / (0.85 * fc' * b)  // عمق المنطقة المضغوطة
φMn = φ * As * fy * (d - a/2) / 10^6  // kN.m

// φ (معامل القوة)
// ECP: φ = 0.95 للشد (tension-controlled)
// ACI: φ = 0.90 للشد
```

### 1.7 تصميم أوراق القص (Shear Design)
**حساب إجهاد القص:**
```kotlin
vu = Vu / (b * d)  // N/mm²
```

**مقاومة الخرسانة للقص (Vc):**
```kotlin
// ECP
Vc = 0.24 * sqrt(fcu) * b * d / 1000  // kN

// ACI
Vc = 0.17 * λ * sqrt(fc') * b * d / 1000  // kN
// λ = 1.0 للخرسانة العادية
```

**حديد القص المطلوب:**
```kotlin
// إذا كان Vu > φVc
Vs = (Vu / φ) - Vc
Av/s = Vs / (fyv * d)  // mm²/mm
```

## 2. معادلات العمود (Column Design)

### 2.1 حساب قدرة التحمل المحورية
**الكود المصري (ECP 203):**
```kotlin
// للأعمدة المربعة والمستطيلة
Pn = 0.67 * fcu * (Ag - As) + fy * As  // N
φPn = 0.75 * Pn / 1000  // kN (تصغير 0.75 للأعمدة)

// للأعمدة الدائرية بالحلزون
Pn = 0.67 * fcu * (Ag - As) + 1.15 * fy * As
φPn = 0.70 * Pn / 1000  // kN
```

**ACI 318-19:**
```kotlin
Pn = 0.85 * fc' * (Ag - As) + fy * As
φPn = 0.65 * Pn / 1000  // kN (للأوراق)
φPn = 0.75 * Pn / 1000  // kN (للحلزون)
```

### 2.2 حساب العزم والانحناء
```kotlin
// معامل الانحناء (K)
K = lu / r  // where r = 0.3h for rectangular

// إذا كان K > 34-12(M1/M2) → عمود طويل (long column)
```

### 2.3 تسليح الأعمدة
```kotlin
// الحد الأدنى للتسليح
ρ_min = 0.01  // 1% من مساحة المقطع

// الحد الأقصى للتسليح
ρ_max = 0.08  // 8% للأعمدة المربعة
ρ_max = 0.10  // 10% للأعمدة الدائرية

// عدد الأسياخ
As_min = 0.01 * b * h  // mm²
As_max = 0.08 * b * h  // mm²
```

### 2.4 تصميم الأوراق (Ties)
```kotlin
// قطر الأوراق
φ_tie ≥ max(6mm, φ_main/4)

// التباعد
s ≤ min(16*φ_main, 48*φ_tie, b)
```

### 2.5 تصميم الحلزون (Spiral)
```kotlin
// نسبة التسليح الحلزوني
ρ_s = 0.45 * (Ag/Ach - 1) * (fc'/fy)

// قطر الحلزون
φ_spiral ≥ 6mm

// التباعد
25mm ≤ s ≤ 75mm
```

## 3. معادلات القاعدة (Footing Design)

### 3.1 حساب أبعاد القاعدة
```kotlin
// حسب ضغط التربة المسموح
A_req = P / q_all  // m²

// للقاعدة المربعة
B = L = sqrt(A_req)

// للقاعدة المستطيلة
B = A_req / L
```

### 3.2 حساب ضغط التربة الفعلي
```kotlin
q_max = P/A + M/S  // kN/m²

// S = section modulus = B*L²/6
```

### 3.3 تصميم قص الثقب (Punching Shear)
```kotlin
// محيط القص
bo = 2*(c1 + c2) + 4*d  // mm (عند d/2 من الوجه)

// إجهاد القص
vu = Vu / (bo * d)  // N/mm²

// مقاومة القص (ECP)
φVc = 0.25 * sqrt(fcu) * bo * d / 1000  // kN

// مقاومة القص (ACI)
φVc = min(
    0.17 * (1 + 2/β) * sqrt(fc') * bo * d,
    0.33 * sqrt(fc') * bo * d
) / 1000  // kN
```

### 3.4 تصميم قص الألواح (One-way Shear)
```kotlin
// عند d من وجه العمود
Vu = q * A_tributary

φVc = 0.16 * sqrt(fcu) * b * d / 1000  // ECP
φVc = 0.17 * sqrt(fc') * b * d / 1000   // ACI
```

### 3.5 تصميم العزوم والتسليح
```kotlin
// العزم عند وجه العمود
Mu = q * L² / 2  // kN.m/m

// d المطلوب
Ru = Mu*10^6 / (1000*d²)
d_req = sqrt(Mu*10^6 / (Ru*1000))

// As
As = Mu*10^6 / (0.87*fy*z)  // mm²/m
```

## 4. معادلات البلاطة (Slab Design)

### 4.1 البلاطة الاتجاه الواحد (One-way Slab)
```kotlin
// عندما يكون Ly/Lx > 2
// يحمل في الاتجاه القصير فقط

// العزم
Mx = w * Lx² / 8  // بسيط
Mx = w * Lx² / 10 // مستمر

// السماكة
h_min = L/25  // ECP (بسيط)
h_min = L/30  // ECP (مستمر)
```

### 4.2 البلاطة الاتجاهين (Two-way Slab)
```kotlin
// عندما يكون Ly/Lx ≤ 2

// طريقة المعاملات (Coefficients Method)
// Marcus أو Grashoff-Rankine

// معاملات العزم
Cx, Cy = f(Ly/Lx, نوع الدعامات)

Mx = Cx * w * Lx²
My = Cy * w * Lx²
```

### 4.3 البلاطة المسطحة (Flat Slab)
```kotlin
// سماكة البلاطة
h_min = Ln / 30  // بدون لوحة إسقاط
h_min = Ln / 33  // مع لوحة إسقاط

// عزم شريحة العمود (Column Strip)
M_col = 0.75 * M_total  // للاتجاه الطويل
M_col = 0.60 * M_total  // للاتجاه القصير

// عزم الشريحة الوسطى (Middle Strip)
M_mid = M_total - M_col
```

## 5. معادلات السلم (Staircase Design)

### 5.1 التحقق من الراحة (Comfort Check)
```kotlin
// قاعدة 2R + T
comfort_check = 2*R + T  // mm

// الحدود المريحة
// 600 ≤ 2R + T ≤ 640 mm

// زاوية الميل
θ = atan(R/T) * 180/π  // degrees
// المريح: 25° ≤ θ ≤ 35°
```

### 5.2 حساب أحمال السلم
```kotlin
// الوزن الذاتي للدرجة
w_step = (R/2 + T) * γ_concrete / 1000  // kN/m²

// الوزن الذاتي للسواعد
w_waist = t * γ_concrete / 1000  // kN/m²

// الحمل الكلي
w_total = 1.4*(w_step + w_waist + finishes) + 1.6*live_load
```

### 5.3 تصميم السواعد
```kotlin
// العزم
M = w * L² / 8  // kN.m/m

// As
As = M*10^6 / (0.87*fy*z)  // mm²/m

// التسليح الموزع
As_dist = 0.0013 * 1000 * t  // mm²/m (ECP)
```

## 9. ملاحظات التنفيذ

### 9.1 التقريبات المستخدمة
```kotlin
// sqrt(0.67) ≈ 0.818
// sqrt(0.85) ≈ 0.922
// π ≈ 3.14159
```

### 9.2 الحدود الدنيا والقصوى
- **نسبة تسليح الكمرة**: 0.0033 ≤ ρ ≤ 0.04
- **نسبة تسليح العمود**: 0.01 ≤ ρ ≤ 0.08
- **تباعد حديد البلاطة**: 100 ≤ s ≤ 250 mm
- **سمك البلاطة**: 100 ≤ h ≤ 300 mm (حسب الSpan)

### 9.3 التحذيرات
1. التطبيق يستخدم fc' وليس fcu للأكواد الأمريكية والسعودية
2. معامل φ يختلف بين الأكواد
3. يجب التحقق من انزلاق الحديد (bond) في التفاصيل
4. يجب التحقق من انبعاج الكمرات الطويلة (L/d > 20)
